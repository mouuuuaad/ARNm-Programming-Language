# ARNm Language Tools (VS Code)

This extension provides:
- Syntax highlighting for `.arnm`
- Snippets for common ARNm constructs
- Keyword/type/builtin completions
- ARNm Forge Dark/Light themes

## Quick Start

1. Open this folder in VS Code.
2. Run `Developer: Install Extension from Location...` and select `editor/vscode`.
3. Open an `.arnm` file to activate.

## Themes

- `ARNm Forge Dark`
- `ARNm Forge Light`

## Notes

Completions are intentionally lightweight and can be expanded with a full LSP later.
